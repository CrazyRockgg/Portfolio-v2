# Portfolio-v2

A modern, responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript. Features a sleek design with dark/light theme toggle and smooth scrolling navigation.

## 🎨 Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Dark/Light Theme**: Toggle between dark and light themes with smooth transitions
- **Modern UI**: Clean, contemporary design with gradient text effects and smooth animations
- **Smooth Navigation**: Anchor-based navigation with smooth scrolling
- **Hamburger Menu**: Mobile-friendly navigation with hamburger menu for smaller screens
- **Hero Section**: Eye-catching landing area with call-to-action buttons
- **Projects Showcase**: Dedicated section to display portfolio projects
- **Contact Section**: Easy way for visitors to get in touch
- **CSS Variables**: Theme management using CSS custom properties for easy customization

## 📁 Project Structure

```
Portfolio/
├── index.html      # Main HTML file with page structure
├── style.css       # Styling with dark/light theme support
├── script.js       # JavaScript for interactive features
└── README.md       # Project documentation
```

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Advanced styling with custom properties and transitions
- **JavaScript (Vanilla)**: DOM manipulation and event handling

## 🚀 Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. No build process or dependencies required - it runs instantly!

## 💡 Features Overview

### Theme Toggle
Click the "Toggle Mode" button in the navigation to switch between dark and light themes. The preference is managed dynamically through CSS class switching.

### Navigation
- Smooth scrolling to different sections (Home, Projects, Contact)
- Responsive hamburger menu for mobile devices
- Theme toggle button in the header

### Sections
- **Hero Section**: Introduction with a call-to-action
- **Projects Section**: Showcase your work and portfolio pieces
- **Contact Section**: Display contact information

## 🎯 Customization

### Color Scheme
Edit the CSS variables in `style.css` under the `:root` selector to change the color theme:

```css
:root {
    --bg-dark: #282828;
    --bg-darker: #222222;
    --text-main: linen;
    --accent: #f1c40f;
    /* ...other variables */
}
```

### Content
Edit the HTML in `index.html` to:
- Change your name and title
- Add your projects
- Update contact information
- Customize navigation links
